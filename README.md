# CareerOS

CareerOS is Marc Agosin's single-user, AI-native job-hunt OS. It keeps a verified facts corpus,
tracks jobs/outreach/resumes, and scans public ATS boards and funding feeds.

The current build covers the corpus, tracker, job scanner, and funding scanner. Resume tailoring,
interviewer, coach, and Obsidian sync are later phases.

## Stack

- Next.js (App Router, TypeScript, Tailwind, shadcn/ui)
- Supabase (Postgres, Auth, RLS) — see `supabase/migrations/`
- Anthropic API for resume tailoring / teardown / interview flows — see `app/api/`

## Getting started

```bash
npm install
cp .env.local.example .env.local   # fill in Supabase + Anthropic credentials
npm run dev
```

Use a hosted Supabase project:

```bash
supabase link
supabase db push
```

Create the single auth user in Supabase Studio, set the environment variables locally and in
Vercel, then deploy. Vercel cron configuration lives in `vercel.json`.

## First use

Drop `marc-facts.md` and a master resume into `docs/`. Import playbooks, create a calibration,
approve target companies, then scan and triage the digest. Facts generated by import remain
proposed until verified in the Inbox.
